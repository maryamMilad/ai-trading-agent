# AI Trading Agent

Automated stock analysis using AI and news sentiment.

## Features
- Daily analysis of top stocks
- AI-powered predictions using Claude
- News sentiment analysis
- Historical tracking

## Architecture
- **GitHub Actions**: Runs daily analysis
- **Supabase**: Stores predictions
- **Render.com**: API server
- **QuantConnect**: Trading execution
